# Docker

Docker and Docker Compose configuration for AniStream deployment.

## Purpose

Run the full stack (frontend, backend, Redis, SQLite volume) with a single command on a homelab Mac Mini.

## Responsibilities

- Multi-service `docker-compose.yml`
- Production Dockerfiles for frontend and backend
- Volume mounts for persistent data

## Status

Will be added once frontend and backend are scaffolded.

## Usage (planned)

```bash
docker compose up
```
